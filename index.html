Moving to HTTPS Guide

Community site to help site owners migrate to HTTPS with a simple yet tried and tested plan. Allowing you to filter the plan based on multiple platforms, hosting envirments and control / access levels.

Are you migrating your site from HTTP to HTTPS? #movingtoHTTPS

Outline

Pre move

  1. Select an SSL certificate

    There are 3 types of SSL/TLS certificates you can setup on your site:

    • DV - Domain Validation (Free or purchase available)
    • OV - Oranization Validation (Requires purchase)
    • EV - Extended Validation (Requires purchase)

  2. Dedicate some time to your migration

    Though installing a certificate can be pretty quick, the actual full migration process can take a good chunk of time to complete

  3. Consider making some changes pre migration

    Some changes such as fixing mixed content can often be done pre migration.
    We've listed it later on as part of the migration flow, but feel free to skip ahead to this step in order to ease the migration later on.
